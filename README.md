# Verilog Fundamentals

This repository contains fundamental digital design implementations using Verilog HDL. It covers logic gates, combinational circuits, and sequential circuits developed using Gate-Level, Dataflow, Structural, and Behavioral modeling techniques.

Each project includes:
- Design Under Test (DUT)
- Testbench
- Simulation verification
- Waveform results where applicable

The repository serves as a learning resource and portfolio for RTL Design, Digital Design, FPGA, and VLSI engineering roles.
